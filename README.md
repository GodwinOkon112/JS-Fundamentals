# ALX Fundamentals Of Javascript Project

## Description

This project focuses on the fundamental of javascript, it focuses mainly on :

1. running javascript in node instead of the browser.
2. variables
3. comments
4. data types
5. loops
6. conditional statement
7. user input
8. arrays

## Installation

1. clone the repository
2. you do not need to run "npm install", since no dependencies were installed, just git bash terminal.
3. you can now run it in th terminal

## Usage

just open the git bash terminal and navigate to the directory where the files are and "node <file_name> to run it on the terminal.
